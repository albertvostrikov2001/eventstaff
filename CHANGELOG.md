# Changelog

## [0.1.0] — 2026-03-30

### Added
- Monorepo structure (pnpm workspaces + Turborepo)
- Docker Compose: PostgreSQL 15, Redis 7, Adminer
- Full Prisma schema with all domain models (25+ models)
- Fastify API server with Prisma, Redis plugins, health check, Swagger
- Next.js 14 App Router with route groups for all 5 zones
- Design system: CSS variables, Tailwind config, typography
- UI components: Button, Card, Input, Textarea, Badge, Avatar, Skeleton, Toast, Modal, RatingStars, EmptyState
- Homepage with hero, categories, stats, CTA sections
- Auth pages: login, register, forgot password
- Dashboard stubs: worker, employer, admin
- Shared package: Zod schemas, constants, utilities, TypeScript types
- README with quick start instructions
